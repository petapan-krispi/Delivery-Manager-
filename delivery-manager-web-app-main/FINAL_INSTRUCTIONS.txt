✅ WHAT'S INCLUDED:

- All original features: Add/Edit/Delete customers, Search, Print labels, Export
- Professional web interface
- Easy to use and update

🚀 HOW TO START:

OPTION 1 - EASIEST (Recommended):
- Windows: Double-click "run_windows.bat"
- Mac: Double-click "run_mac.sh" or run: ./run_mac.sh
- Linux: Run: ./run_linux.sh

OPTION 2 - Python launcher:
- Run: python launch.py

OPTION 3 - Manual:
- Install: pip install -r requirements_web.txt
- Run: streamlit run app.py

🌐 The app will open at: http://localhost:8501

🔧 IF YOU HAVE ISSUES:

1. FIRST: Run the test script
   - Run: python test_app.py
   - This will check if everything is working

2. IF SESSION ENDS QUICKLY:
   - Run: debug_run.bat (Windows)
   - Check error messages
   - Make sure Python is installed correctly

3. IF PACKAGES FAIL TO INSTALL:
   - Check internet connection
   - Try running as Administrator (Windows)
   - Try: pip install --user -r requirements_web.txt

4. IF BROWSER DOESN'T OPEN:
   - Manually go to: http://localhost:8501
   - Make sure no other app is using port 8501
